These are the basics of Javascript (JS).

JS is a loosely typed language. This means that you can assign and reassign data.


Variables & Constants

Variables are used to create and store data.
A variable must be declared in order for JS to recognize it and be usable.

The are three ways to declare a variable.

  • var
  • let
  • const

var is the oldschool way of declaring a variable. Use the modern approaches let and const instead.

let allows you to declare a variable whos value can change.

const allows you to declare a variable whos value cannot change.


Data Types

There are only seven data types in JS.

Any data type can be used with a variable.

  • strings
  • numbers
  • boolean
  • null
  • undefined
  • object
  • symbol
